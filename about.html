About this tool

                   This tool is created for the educational purpose of teaching beginners to create their own data visualization tools from a computer system or other data source.

                    This tool is purely demonstrative allowing the user to learn how most of these data viewing tools are made. At the same time, I've tried to explain the source code as well as possible to help beginners create their own visualization elements. I have also included a "Hello World" example to encourage learners to explore this method of building web applications.

                   Also this application is based on GNUPLOT for generating graphical data and I have included a CSV to HTML (code that I just integrated in the application and was originally created by derekeder).

                  Now you have the opportunity to learn how to make such web applications and build your own reports about the system. Of course, other automations are possible and I will demonstrate them in later versions of this application.

                  This tool intends to be a "bare-bone" that system administrators can alter for their own needs, the application containing several examples applicable to various functionalities.

         

    Home Page:

                        Contains a CGI hat reports on overall system status and demonstrate how you can run a system command, grep the output into a variable and pass it to the html page.
         

    Networking Page:

                        Contains a CGI that shows the traffic on ethernet and wifi card.                   Also runs a simple python script to check DNS servers and display the script output.                   Use this page in order to manipulate the output from external scripts.
         

    Security Report Page:

                        Contains the CSV to HTML tool made by derekeder and is used to see security logs, esspecially from Snort                   Use this page to display security alerts or any other csv file
         

    Graphics Page:

                        Contains a simple html page that dysplay graphics made by GNUPLOT. Use this tool to generate you graphis from most data files.
         

    Sample Page:

                        Contains a "Hello World" example of a CGI script to help you learn and practice.
